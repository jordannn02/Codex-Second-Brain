## Demo Flows

1. Debugging route learning: failed broad search becomes a suppressed path, and graph-first tracing becomes preferred.
2. Tool overload auto-router: the agent classifies the task and chooses the right capability without asking the user to pick tools.
3. Meeting-to-execution memory: decisions, owners, risks, and repeated themes become durable execution memory after delivery.
