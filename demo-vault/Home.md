# Home

## For future Claude

This dashboard is intentionally small. It points to the demo workflow that shows how user work becomes a delivered artifact plus a reusable capture.

## Demo Flow

1. Read startup files.
2. Classify the request.
3. Deliver the Markdown artifact.
4. Capture only reusable workflow evidence.
5. Leave broad cleanup for weekly consolidation.
